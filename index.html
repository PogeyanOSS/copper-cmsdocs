Copper CMS Server

CMIS Server implement based on Apache Chemistry/OpenCMIS.

View on GitHub Download .zip

Copper CMS project implements the OpenCMIS specifications. It implements the following CMIS services,

  • Repository Service It used to discover information about the repository, including information about the repository and the object-types deﬁned for the repository. Furthermore, it provides operations to create, modify and delete object-type deﬁnitions if that is supported by the repository. It supports following methods.

  • Navigation Service It used to traverse the folder hierarchy in a CMIS repository, and to locate documents that are checked out. It supports following methods.

  • Object Service CMIS provides id-based CRUD (Create, Retrieve, Update, Delete) operations on objects in a repository. It supports following methods.

  • Discovery Service It used to search for query-able objects within the repository. It supports following methods.

  • Versioning Service It used to navigate or update a document version series. It supports following methods.

  • Relationship Service It used to retrieve the dependent relationship objects associated with an independent object. It supports following methods.

  • Policy Service It used to apply or remove a policy object to a controllablePolicy object. It supports following methods.

  • ACL Service It used to discover and manage Access Control Lists.It supports following methods.
